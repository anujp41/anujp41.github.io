CaptAR

Real-time mobile game built with React-Native, React-Native-Maps, Geolib, Redux, Firebase for iOS and Android

This game was put together for the Fullstack Academy Capstone Project. I worked on it with 3 of my classmates to build the game in 2.5 weeks. We had a lot of fun working on this project and managed to build a great game while learning and researching a lot of cool topics and technologies (geolocation/geofencing) that we were not aware of preiously.

Remainder app built with React Native, Redux, React-Native-Maps

App allows to set remainders that allows you to leave notes for yourself to visit a particular place. Currently working to add geofencing that will send remainders based on how close you are to a certain 'virtuanote'.
